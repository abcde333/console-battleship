# Battleship (Морской бой)

Консольная реализация игры «Морской бой» на Python.

Проект позволяет играть человеку против бота.
Игра полностью проходит в консоли и отображает игровые поля в текстовом виде.
Все изменения состояния игры (корабли, выстрелы, попадания и промахи) сохраняются в файлы в папке `data`.

---

## Структура проекта

```
python_proj/
│
├── main.py                  # Точка входа в игру
├── src/
│   ├── utils.py              # Вспомогательные функции (координаты, соседи и т.д.)
│   ├── player_input.py       # Ввод кораблей игрока
│   ├── bot_generation.py     # Генерация кораблей бота
│   ├── gameplay.py           # Игровая логика
│   └── board.py              # Печать игровых полей
│
├── data/
│   ├── player_ships.csv      # Корабли игрока
│   ├── bot_ships.csv         # Корабли бота
│   ├── player_shots.csv      # Выстрелы игрока
│   └── bot_shots.csv         # Выстрелы бота
│
├── requirements.txt
└── README.md
```

---

## Запуск игры

1. Установите Python версии **3.8 или выше**
2. Установите зависимости:

   ```bash
   pip install -r requirements.txt
   ```
3. Запустите игру:

   ```bash
   python3 main.py
   ```

---

## Правила игры

* Используется классическое поле **10×10**
* Корабли не могут соприкасаться, даже по диагонали
* Обозначения на поле:

  * `O` — корабль
  * `H` — попадание
  * `M` — промах
  * `.` — пустая клетка
* Побеждает тот, кто первым уничтожит все корабли противника

---

## Хранение данных

После каждого хода:

* выстрелы игрока и бота сохраняются в CSV-файлы
* состояние игры обновляется не только в консоли, но и в папке `data`

Это позволяет восстановить игру или проверить корректность логики.
